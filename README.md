# Compilation
- In order to compile this project the sqlite amalgamation files are needed place them into the sqlite directory.
### file structure
```
---root
   ---sqlite
      ---(Put Your SQLite amalgamation files here!)
   ---build
      ---(CMake cache and build in this directory!)
   ---include
   ---src
```
